요청하신 OCR API 는 정상 작동하지 않았습니다. 요청 파라미터를 확인하고 다시 시도해 주세요.

원인은 400 Bad Request 오류로, 서버가 요청을 이해할 수 없거나 잘못된 요청을 받았기 때문입니다. 이 오류는 클라이언트 쪽에서 잘못된 요청을 보냈을 때 발생하는데, 주로 파라미터가 잘못 전달되었거나 누락되었을 경우 발생합니다.

요청 파라미터를 확인하는 가장 일반적인 방법은 API 문서를 참고하는 것입니다. API 문서는 요청에 필요한 파라미터와 그 값을 알려주는데, 이를 잘 지켜야만 정상적인 응답을 받을 수 있습니다. 또한 API 문서에는 허용되지 않는 파라미터를 넘겼을 때 발생하는 오류도 설명되어 있기 때문에, 반드시 확인해야 합니다.

또한 API 요청을 보낼 때는 다른 요청 데이터와 함께 유효한 요청 데이터를 보내는 것이 중요합니다. 예를 들어, 이미지 데이터를 OCR API 에 요청하고 있었다면, 이미지 데이터가 정상적으로 전송되고 있는지도 확인해야 합니다. 파라미터가 문제가 아닌 경우, 다른 요청 데이터가 문제가 되는 경우도 있기 때문입니다.

요청 파라미터와 데이터를 모두 확인했는데도 오류가 발생하는 경우에는 API 제공사에 문의하는 것이 좋습니다. API 제공사는 오류 메시지를 통해 어떤 부분에서 오류가 발생했는지 알려주고, 조치 방법을 안내해줄 수 있습니다.
